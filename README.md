# Starter de sitios BOOST SALES LAB

Base para landings y sitios institucionales en React, Vite, TypeScript, Tailwind y Vercel.

## Inicio

1. Copiar desde `WEB-SYSTEM` con el script de alta.
2. Completar los cinco documentos de estrategia del proyecto.
3. Instalar dependencias: `npm install`.
4. Ejecutar: `npm run dev`.
5. Implementar con Codex siguiendo `AGENTS.md`.

Antes de publicar, reemplazar todos los valores `PENDIENTE CLIENTE`, actualizar title y meta descripcion en `index.html`, y configurar el enlace real de WhatsApp en `src/content/site.ts`.

## Vercel

Crear un proyecto Vercel nuevo para cada cliente y conectar su repositorio. Las variables secretas se cargan en Vercel; no se escriben en archivos versionados.
